# final-attendance-app

## アプリ概要

Spring Boot を使用したタスク管理アプリです。

ユーザー登録、ログイン、
タスク登録、一覧表示、編集、削除機能を実装しています。

---

## 使用技術

- Java17
- Spring Boot 3.2.5
- Thymeleaf
- MyBatis
- PostgreSQL
- HTML/CSS
- Git/GitHub

---

## 機能一覧

- ユーザー登録
- ログイン機能
- タスク一覧表示
- タスク登録
- タスク編集
- タスク削除
- ページング機能

---

## 画面一覧

- ホーム画面
- ログイン画面
- ユーザー登録画面
- タスク一覧画面
- タスク登録画面
- タスク編集画面

---

## DB

PostgreSQL を使用。

users テーブルと tasks テーブルを利用しています。
